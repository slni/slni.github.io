HUGO快速入门

一、简单介绍

Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

hugo源码地址：https://github.com/gohugoio/hugo

hugo存在两个版本，标准版和扩展版。扩展版有如下额外的功能：

  • Encode WebP images (you can decode WebP images with both editions)
  • Transpile Sass to CSS using the embedded LibSass transpiler

我们建议安装扩展版。

二、安装hugo

macOS可以使用homebrew进行安装，homebrew安装的就是hugo的扩展版本。如果您是其它操作系统，可以参考官方安装文档：https://gohugo.io/installation/

brew install hugo

安装成功后，查看版本号进行验证。

hugo version

三、新建站点

使用Hugo快速生成站点，比如希望生成到 /path/to/site 路径：

hugo new site /path/to/site

这样就在 /path/to/site 目录里生成了初始站点，进去目录：

cd /path/to/site

站点目录结构如下：

  • config.toml: 网站配置文件
  • content目录：文章的存放目录
  • public目录：生成的静态网站存在目录
  • static目录：资源文件，图片等公共资源存放目录
  • themes目录：相关主题的下载存放目录

四、添加文章

  1. 创建一个 about 页面：
// 站点根目录运行
hugo new about.md
  1. 创建第一篇文章，放到 post 目录，方便之后生成聚合页面。
// 站点根目录运行
hugo new post/first.md

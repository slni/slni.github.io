HUGO部署-GitHub Pages

Anonymous 收录于 hugo
 2023-03-04  约 369 字  预计阅读 1 分钟 

前言：

hugo生成好的静态网页可以使用多种方式部署到线上，以下使用GitHub Pages进行部署。

第一步：创建仓库

首先在GitHub上创建一个Repository，命名为：slni.github.io （slni替换为你的github用户名）。

第二步：生产静态页面到public目录

修改config.toml中的baseUrl的值，baseURL = 'https://slni.github.io/' 在站点根目录执行 Hugo 命令生成最终页面：

$ hugo

注意，这里的baseUrl使用https,不用使用http, 不然可能导致css访问失效的问题。 所有静态页面都会生成到 public 目录。

第三步：将public目录关联到远程仓库，并上传静态页面

$ cd public
$ git init
$ git remote add origin https://github.com/slni/slni.github.io.git
$ git add -A
$ git commit -m "first commit"
$ git push -u origin main     // 不是master分支，具体看你github对应的仓库

浏览器里访问：https://slni.github.io/
